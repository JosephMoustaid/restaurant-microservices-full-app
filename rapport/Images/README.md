## Comment prendre les captures

### Pour l'application web :
1. Démarrez tous les services backend
2. Démarrez le frontend React
3. Naviguez dans l'application
4. Utilisez F12 (Chrome DevTools) pour changer la taille de fenêtre
5. Prenez des captures avec Shift+Windows+S ou un outil de capture

### Pour les diagrammes :
- **Architecture globale** : Utilisez draw.io ou Lucidchart
- **Diagramme de séquence** : PlantUML ou draw.io
- **Diagramme de déploiement** : draw.io ou Visual Paradigm

## Dimensions recommandées

- Captures web : 1920x1080 ou 1440x900
- Diagrammes : Format vectoriel (SVG) ou PNG haute résolution
- Compression : Optimiser les images avec TinyPNG ou ImageOptim
